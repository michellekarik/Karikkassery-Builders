# Karikkassery-Builders
Website for Karikkassery Builders, a local family-run home construction business based in Kochi, Kerala, India. This project was developed independently to create a simple online presence showcasing the homes and construction services offered by the family business.

Tech Stack
- HTML5
- CSS3
- JavaScript
- Vercel for deployment
- GitHub for version control
